Short-Term (0–30 Days)

  1. Home safety upgrades (grab bars, lighting, remove trip hazards)
  2. Pharmacy blister packs + locked dispenser + med log
  3. PCP follow-up + med reconciliation + PT referral
  4. Start home care aide 20–25 hrs/week
  5. Family command center (shared calendar + one-pager emergency contacts)

Mid-Term (30–90 Days)

  1. Structured routines (hydration, meals, movement, sleep)
  2. Adult day health 1–2 days/week
  3. Cognitive + mood reassessment
  4. Care-hour re-evaluation every 30 days
  5. Legal document review (proxy, POA, HIPAA, MOLST discussion)

Long-Term (3–12 Months)

  1. Pathway: standard home care → enhanced home care → memory care if required
  2. Trigger thresholds monitored monthly
  3. Benefits + financing strategy review (VA/LTC/MassHealth consult)
  4. Quarterly care conference with PCP + care manager + family
